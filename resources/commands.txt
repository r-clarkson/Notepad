
COMMAND NAME  |  COMMAND FUNCTION  |  COMMAND OPTIONS

 add -x           add to notebook      -t (add a note by typing text)
                                       -d (add a note through dictation)
                                       -i (add a type of identifier to an existing note)

COMMAND NAME |   COMMAND FUNCTION  |  COMMAND OPTIONS

 delete -x     delete from notebook     -n (delete a note and its identifiers by notename)
                                        -i (delete a specific identifier)

COMMAND NAME |  COMMAND FUNCTION  |  COMMAND OPTIONS

 search -x     search in notebook     -w (returns the filenames of notes containing an entered identifier)
                                      -a (returns all notes which contain one+ identifiers of a certain type)
                                      -o (returns notes organized by identifiers of a certain type)
                                      -l (returns list of all identifiers of a certain type)
                                      -t (returns notes organized in topological order)

COMMAND NAME  |  COMMAND FUNCTION

 filepath         give notes location via filepath

COMMAND NAME  |  COMMAND FUNCTION

 quit             terminate program
